A vibrant, atmospheric, and responsive promotional landing page for a surfing school.

**[👀 Live Preview](https://surfing-school-tenteros.netlify.app/)**

### 📝 About the Project
This repository showcases a modern front-end landing page dedicated to a surfing school. The main goal of the project is to capture the freedom and energy of the sport through clean layout design and smooth native interactions, built entirely without heavy frameworks.

### 🚀 Key Features
* **Vanilla Stack:** Developed using pure HTML5, CSS3, and basic JavaScript.
* **Responsive Design:** Fully optimized to ensure a seamless experience across desktops, tablets, and mobile screens.
* **Smooth UX:** Includes fluid animations, interactive elements, and crisp typography to deliver an engaging user experience.

## 🇷🇺 Описание на русском языке

Яркий, атмосферный и адаптивный промо-лендинг для школы серфинга.

**[👀 Смотреть Live Preview](https://surfing-school-tenteros.netlify.app/)**

### 📝 О проекте
В этом репозитории представлена современная верстка страницы для школы серфинга. Главная цель проекта — передать атмосферу свободы и драйва через чистый дизайн и нативную интерактивность, созданную без использования тяжелых фреймворков и сторонних библиотек.

### 🚀 Ключевые особенности
* **Чистый стек:** Проект реализован исключительно на HTML5, CSS3 и базовом JavaScript.
* **Адаптивность:** Интерфейс полностью оптимизирован для корректного отображения на ПК, планшетах и мобильных экранах.
* **Плавный UX:** Реализованы плавные анимации, интерактивные элементы и четкая типографика для максимального погружения пользователя.
